# 📊 R Basic Manual

A step-by-step learning repository for mastering the fundamentals of **R programming**,
from basic syntax to real-world data analysis and data cleaning.

---

## 🚀 Overview

This repository is designed as a **hands-on learning guide** for R.

Instead of just learning syntax, it focuses on:

* understanding data
* performing analysis
* visualizing results
* building practical skills

---

## 🧭 Learning Flow

```text
01 Hello R  
02 Basic Syntax  
03 Vectors  
04 Data Frame  
05 Basic Statistics  
06 Visualization  
07 Real Data Analysis  
08 Data Cleaning  
```

Each chapter includes:

* 📘 `.Rmd` → explanation
* 💻 `.R` → hands-on practice

---

## 📂 Project Structure

```text
R-Basic-Manual/
```
